== Description ==

Netuy Email Marketing es un plugin utilizado para agregar subscriptores al emailmarketing de Netuy a través de formularios generados por los plugins de formularios mas populares de Wordpress,  Contact form 7, WPForms y Elementor. 

= Docs and support =

You can find https://netuy.net

= Privacy notices =

This plugin, in itself, does not:

* track users by stealth;
* write any user personal data to the database;
* use cookies.

If you activate certain functions in this plugin, the personal data of the sender of the contact form will be sent to the account that you associated with netuy emailmarketing

= Recommended plugins =

The following plugins are recommended for Netuy Emailmarketing users:

* [Contact form 7](https://es.wordpress.org/plugins/contact-form-7/) 
* [WPFroms](https://es.wordpress.org/plugins/wpforms-lite/) 
* [Elementor](https://es.wordpress.org/plugins/elementor/)


== Installation ==

1. Upload the entire `netuy_emailmarketing` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

For basic usage, have a look at the [netuy website](https://netuy.net/).
